Interface RepositoryService

All Known Implementing Classes:
RepositoryServiceDbData
public interface RepositoryService

This interface contains methods that are related to accessing/creating/deleting... documents, Users, and Projects for the annotation system. while meta data about documents and projects and users are stored in the database, source and annotation documents are stored in a file system

Author:
Seid Muhie Yimam, Richard Eckart de Castilho
