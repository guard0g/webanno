public class BratAnnotationDocumentVisualizer
extends BratVisualizer

Displays a BRAT visualisation and fills it with data from an AnnotationDocument. We do not use a CAS as model object because the CAS is large and does not serialize well. It is easier to drive this component using a reference to the CAS (here an AnnotationDocument) and let the component fetch the associated CAS itself when necessary.
