          Description

Class Summary
Argument The Arguments used during arc annotation in the form of [["Arg1","p_21346"],["Arg2","p_21341"]] to denote a given arc annotation such as dependency parsing and coreference resolution
Entity An Entity is a span annotation with id, type and offsets of the span annotation of the format ["p_22406","KON",[[1125,1128]]] - id of POS 22406, of type KON and start offset=1125, end offset = 1128
EntityType Different attributes of an Entity used for its visualisation formats.
Offsets The start and End offset positions of a span annotation as required by the Brat protocol
Offsets.OffsetsDeserializer Deserialize Offsets from JSON to Java.
OffsetsList List of Offsets.
Relation A relation between span annotations -> an arc annotation.
RelationType Type of an arc.
Stored Used to return name and suffix of annotation document exports.
